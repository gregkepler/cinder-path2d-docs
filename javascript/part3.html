Part 3: Advanced Functions

Advanced functions blah blah blah

  • calcBoundBox(): .
  • calcPreciseBoundBox(): .
  • contains(): Accepts 1 coordinate and draws a line from the current point in the path to the given one. Results in a LINETO segment.
  • transformCopy(): Accepts 1 central handle point and 1 end point to create a quadratic bezier curve resulting in a QUADTO segment.
  • subdivide(): Accepts 2 handle points and 1 end point to create a cubic bezier curve resulting in a CUBICTO segment
  • Path2dCalcCache: Accepts a center point, a radius, start angle (in radians), end angle (in radians), and optional boolean for direction of the arc (defaults to true for forward). This results in a CUBICTO segment.
